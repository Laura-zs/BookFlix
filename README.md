# 📚 Biblioteca Online (Bookflix)

Uma aplicação web desenvolvida com React que simula uma interface estilo Netflix para explorar livros. O projeto permite visualizar coleções, acessar detalhes de cada livro e navegar por uma interface moderna e interativa.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🔀 React Router DOM

---

## 🎯 Funcionalidades

* 📚 Listagem de livros em estilo Netflix (scroll horizontal)
* 🔍 Página de detalhes de cada livro
* ⭐ Sistema de avaliação com estrelas
* 📝 Exibição de sinopse
* 📍 Links para encontrar o livro (Amazon, Google Books)
* 🎨 Interface moderna com animações e hover effects

---

## 🖥️ Preview

Interface inspirada em plataformas de streaming:

* Layout dark mode
* Cards com animação
* Hover com título do livro
* Página de detalhes completa
